# Codebook

Use these tags in note metadata, reading logs, annotations, and project files.

## Core Linguistic Areas

| Code | Area |
| --- | --- |
| `PHON` | Phonetics and phonology |
| `MORPH` | Morphology |
| `SYN` | Syntax |
| `SEM` | Semantics |
| `PRAG` | Pragmatics |
| `SOCIO` | Sociolinguistics |
| `HIST` | Historical linguistics |
| `TYPO` | Linguistic typology |
| `FIELD` | Fieldwork and elicitation |
| `CORPUS` | Corpus linguistics |
| `L2` | Second-language acquisition |
| `DISCOURSE` | Discourse analysis |

## Optional Fine Tags

Use dot notation when detail helps:

| Code | Meaning |
| --- | --- |
| `PHON.vowel` | Vowels |
| `PHON.tone` | Tone |
| `MORPH.inflection` | Inflection |
| `MORPH.derivation` | Derivation |
| `SYN.case` | Case |
| `SYN.word-order` | Word order |
| `SEM.aspect` | Aspect |
| `SEM.modality` | Modality |
| `PRAG.politeness` | Politeness |
| `SOCIO.register` | Register |
| `CORPUS.annotation` | Corpus annotation |
