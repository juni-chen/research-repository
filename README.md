# Linguistics Research Repository

This repository stores reusable notes, references, data records, analysis, and project materials for linguistic learning and research.

## Structure

| Folder | Purpose |
| --- | --- |
| `_meta/` | Repository map, conventions, codebook, and glossary. |
| `inbox/` | Fast capture for unsorted notes, ideas, examples, and questions. |
| `notes/` | Refined reusable knowledge: concepts, methods, languages, and paper notes. |
| `references/` | Bibliography, reading log, and source notes. |
| `data/` | Data records, annotation notes, and links to external datasets. |
| `projects/` | Research projects with their own questions, notes, analysis, and outputs. |
| `analysis/` | Shared scripts, notebooks, and small utilities. |
| `outputs/` | Papers, presentations, figures, tables, and other finished artifacts. |
| `templates/` | Reusable note and project templates. |

## Basic Workflow

1. Capture quickly in `inbox/`.
2. Refine reusable ideas into `notes/`.
3. Track sources in `references/`.
4. Keep project-specific work in `projects/`.
5. Put generated or polished materials in `outputs/`.

## Metadata Pattern

Most reusable notes should start with a small metadata block:

```markdown
---
title: Evidentiality
type: concept
status: draft
tags: [SEM, PRAG, MORPH]
languages: [Turkish, Quechua, Japanese]
related: [modality, tense-aspect-mood]
---
```

See `_meta/codebook.md` for the tag system and `_meta/conventions.md` for naming rules.
